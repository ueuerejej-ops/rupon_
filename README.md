My Language Experimental Compiler Project
Idea of the language

The main goal is not just to create another programming language but to learn and experiment with compiler design

Later I want to explore ideas like

different memory management models
greater control over how data is stored
flexible language design
Examples
Hello world style
func main() {
    str hello = "hello"
}
Variables
func main() {
    int a = 10
    int b = 20
}
Strings
func main() {
    str name = "rupon"
}
Function with arguments
func add(int a, int b) {
    int result = a + b
    return result
}
Function call
func main() {
    add(10, 20)
}
Mixed example
func test() {
    int x = 0
    str hello = "hello"
    return x
}
Contributing

This project is still very early in development and I am currently working on it alone

I would appreciate help and feedback from other people

If you want to contribute you can

suggest ideas for language design
help improve parsing or LLVM generation
propose new features
give feedback about syntax and structure

Even small feedback is helpful this is a learning project

Important

This project is not stable yet

A lot of things may change in the future

syntax
features
internal structure

Everything is experimental
